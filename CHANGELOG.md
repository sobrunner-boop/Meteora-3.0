# CHANGELOG - Meteora Order 3.0 Final

## 3.0 Final
- Komplett neue schlanke Version für reine Bestellaufnahme erstellt.
- Bereiche Innen, Außen, Außer Haus und Saal mit den gewünschten Tischnummern angelegt.
- Admin mit PIN-Schutz ergänzt.
- Hauptgruppen, Untergruppen und Artikelverwaltung ergänzt.
- Info-Button pro Artikel ergänzt.
- Freie Bemerkung pro Artikel ergänzt.
- Schnellbemerkungen bewusst entfernt.
- Sonstiges als freier Eintrag ergänzt.
- Design-Einstellungen für Farben, Schriftgröße und Tischgröße ergänzt.
- Backup/Import ergänzt.
- PWA-Dateien für GitHub Pages ergänzt.
